Plugin to do list.

== Description ==

After installing and activating the plugin, a "My to do app" page is created along with a shortcode displaying all created tasks and a form that allows adding new tasks.
The site is available only to logged users and displays the tasks created by the user.
To add a task, enter the task name and click enter button.
Editing the task name is done by clicking on the task and entering a new name and confirming it with the enter button.
To delete a given task, hover over the task and click the appearing X.
